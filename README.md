This repsitory creates the metapackages for MamoLinux. The following metapackages are provided:
1. mamolinux-minimal
2. mamolinux-standard
3. mamolinux-desktop-minimal
4. mamolinux-desktop
5. mamolinux-desktop-full
6. mamolinux-live
